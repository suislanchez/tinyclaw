What's New in TinyClaw

Streaming TUI

Beautiful ratatui terminal interface with real-time SSE token streaming, markdown rendering, and auto-scrolling.

Parallel Tools

Multiple tool calls execute concurrently via tokio::spawn instead of sequentially. Results collected in order.

Token Tracking

Live cost tracking across all providers with atomic counters. See tokens, requests, and estimated USD in real-time.

Session Persistence

Conversations auto-save to disk as JSON. Resume, list, and export sessions with slash commands.

Slash Commands

/help, /cost, /clear, /model, /sessions, /export, /quit. Control everything from the TUI input bar.

Tiered Builds

Three feature tiers: tiny (3.5MB CLI), standard (+TUI), full (+gateway, daemon, channels, OTel). Build what you need.
